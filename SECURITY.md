# Security Policy

## Scope

This repository is a public professional profile and cybersecurity portfolio
index. It does not intentionally contain production secrets, credentials,
customer data, or live third-party target information.

## Reporting a Concern

If you identify exposed credentials, sensitive personal data, unsafe lab
material, or inaccurate security claims in this repository, open a GitHub issue
with a concise description and the affected file path.

Do not include active secrets, exploit payloads for real systems, or private
data in public issues. If sensitive details are required, provide only enough
context to reproduce the concern safely.

## Handling Standard

Reports are triaged using the following order:

1. Evidence preservation
2. Scope confirmation
3. Impact assessment
4. Corrective action
5. Public documentation update, when appropriate

## Intended Use

The content is intended for portfolio review, professional learning, and
audit-friendly cybersecurity documentation. It is not an authorization to test
systems owned by third parties.
